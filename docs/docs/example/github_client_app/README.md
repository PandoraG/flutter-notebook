# github_client_app

```
 - lib
    - asserts   //静态资源
        - fonts      //字体
        - icons      //图标
        - imgs       //图片
    - common    // 一些工具类，如通用方法类、网络接口类、保存全局变量的静态类等
    - I10n      // 国际化相关的类都在此目录下
    - models    //Json文件对应的Dart Model类会在此目录下
    - states    //跨组件共享的状态类
    - routes    //路由配置
    - widgets   //自定义组件库
```
