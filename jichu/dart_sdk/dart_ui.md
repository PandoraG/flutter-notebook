# dart:ui 库

Flutter 应用程序的内置类型和核心原语。

使用：

```dart
import 'dart:ui'
```

该库提供 Flutter 框架 用于引导应用程序（bootstrap applications）的最低级别的服务，例如：用于驱动输入，图形文本，布局和渲染子系统的类。
